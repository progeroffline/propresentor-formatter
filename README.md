# propresentor-formatter

The tool to fast format lyrics to format which you can import to propresentor and get already formatter sliders and grouped by sections

Built with React, TypeScript, Vite, and shadcn/ui.

## Adding components

To add components to your app, run the following command:

```bash
npx shadcn@latest add button
```

This will place the ui components in the `src/components` directory.

## Using components

To use the components in your app, import them as follows:

```tsx
import { Button } from "@/components/ui/button"
```
