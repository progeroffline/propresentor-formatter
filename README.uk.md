## Навіщо

Перенесення тексту пісні в ProPresenter зазвичай означає: вручну передрукувати назви секцій, розбити текст на слайди по одному, і почистити випадкові посилання чи розділові знаки, які хтось залишив у тексті. Цей інструмент робить усе це за одну вставку: закиньте текст у будь-якому вигляді, в якому він трапився під руку — з сайту з текстами пісень, з групового чату, з PDF — і отримайте на виході чистий текст, розмічений за секціями, з одним рядком на слайд, готовий для вставки прямо в редактор аранжування ProPresenter.

## Можливості

**Автоматичне визначення секцій** — розпізнає назви секцій (Verse/Куплет, Chorus/Приспів, Bridge/Бридж та інші) англійською, російською та українською, у будь-якому вигляді, в якому вони трапляються:

- Номер до _або_ після назви — `Куплет 2`, `2 Куплет`, `Verse (2)`, `Verse II`
- Маркери повтору відкидаються як оформлення, а не як зміст — `Chorus x2`, `Приспів х2`
- Поширені одруки та артефакти змішаної розкладки клавіатури — наприклад, `Teг` (латинські _T_/_e_ + кириличне _г_)
- Працює навіть **без порожніх рядків** між секціями — заголовки розпізнаються порядково, а не лише на початку абзацу, тож суцільний, без прогалин, шматок вставленого тексту теж коректно розіб'ється на секції

**Один рядок — один слайд** — кожен рядок тексту пісні стає окремим слайдом ProPresenter, як це зазвичай влаштовано в аранжуваннях ProPresenter.

**Опційне очищення**, застосовується до кожного слайда:

- Велика перша літера
- Видалення розділових знаків (коми, лапки, три крапки та інші)
- Видалення посилань `http(s)://` і `www.`

**Автоматичний титульний слайд** — початковий непозначений блок (зазвичай назва пісні та автор) отримує префікс `Title: ` у скопійованому результаті, але лише якщо в пісні є хоча б одна розпізнана секція — так звичайний шматок тексту не буде розмічено помилково.

**Кольоровий живий перегляд** — кожен слайд відображається окремою карткою в панелі виводу, з рамкою, забарвленою в колір секції (Verse — сині відтінки, Chorus — малиновий/рожево-червоний, Bridge — фіолетовий і так далі — див. [повну таблицю](#підтримувані-секції)), і назвою секції на кожній картці. Достатньо одного погляду, щоб зрозуміти, чи правильно пісня розбилася на секції.

**Керування з клавіатури** — `Ctrl+Enter` для форматування, `Ctrl+Shift+C` для копіювання результату.

**Світла й темна тема**, інтерфейс російською та українською (про те, чому тут узагалі немає сервера — див. [Приватність](#приватність)).

## Опції форматування

Три опційні перемикачі, застосовуються до кожного слайда перед розбиттям:

| Опція                        | Що робить                                                                                                                                                       |
| ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Великі літери**            | Робить великою першу літеру кожного слайда (з урахуванням кирилиці — обробляє й `і`/`ї`/`є`/`ґ`).                                                               |
| **Прибрати розділові знаки** | Прибирає коми, крапки, лапки, `«»`, три крапки й подібні знаки; рядок, що став порожнім після очищення, відкидається повністю, а не лишається порожнім слайдом. |
| **Прибрати посилання**       | Прибирає з рядка посилання `http(s)://` та `www.`; така ж поведінка з відкиданням порожнього рядка.                                                             |

## Швидкий старт

Знадобиться [Node.js](https://nodejs.org/) та npm.

```bash
git clone git@github.com:progeroffline/propresentor-formatter.git
cd propresentor-formatter
npm install
npm run dev
```

Відкрийте виведену локальну адресу, вставте текст пісні та натисніть `Ctrl+Enter`.
