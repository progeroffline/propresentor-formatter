## Зачем

Перенос текста песни в ProPresenter обычно означает: вручную перепечатать названия секций, разбить текст на слайды по одному, и почистить случайные ссылки или знаки препинания, которые кто-то оставил в тексте. Этот инструмент делает всё это за одну вставку: закиньте текст в любом виде, в котором он попался под руку — с сайта с текстами песен, из группового чата, из PDF — и получите на выходе чистый текст, размеченный по секциям, с одной строкой на слайд, готовый для вставки прямо в редактор аранжировки ProPresenter.

## Возможности

**Автоматическое определение секций** — распознаёт названия секций (Verse/Куплет, Chorus/Припев, Bridge/Бридж и другие) на английском, русском, украинском или немецком, в любом виде, в котором они встречаются:

- Номер до _или_ после названия — `Куплет 2`, `2 Куплет`, `Verse (2)`, `Verse II`
- Маркеры повтора отбрасываются как оформление, а не как содержание — `Chorus x2`, `Припев х2`
- Распространённые опечатки и артефакты смешанной раскладки клавиатуры — например, `Teг` (латинские _T_/_e_ + кириллическая _г_)
- Работает даже **без пустых строк** между секциями — заголовки распознаются построчно, а не только в начале абзаца, так что сплошной, без пропусков, кусок вставленного текста тоже корректно разобьётся на секции

**Одна строка — один слайд** — каждая строка текста песни становится отдельным слайдом ProPresenter, как это обычно устроено в аранжировках ProPresenter.

**Опциональная очистка**, применяется к каждому слайду:

- Заглавная первая буква
- Удаление знаков препинания (запятые, кавычки, многоточия и другие)
- Удаление ссылок `http(s)://` и `www.`

**Язык вывода** — выберите, на каком языке будут названия секций (и префикс `Title:`) в результате: 🇬🇧 английский, 🇷🇺 русский, 🇺🇦 украинский или 🇩🇪 немецкий — независимо от того, на каком языке названы секции во вставленном тексте. По умолчанию — английский. Можно вставить песню с русскими заголовками секций, а на выходе получить немецкие — или любое другое сочетание.

**Автоматический титульный слайд** — начальный неразмеченный блок (обычно название песни и автор) получает переведённый префикс `Title: ` в скопированном результате, но только если в песне есть хотя бы одна распознанная секция — так обычный кусок текста не будет размечен по ошибке.

**Цветной живой предпросмотр** — каждый слайд отображается отдельной карточкой в панели вывода, с рамкой, окрашенной в цвет секции (Verse — синие тона, Chorus — малиновый/розово-красный, Bridge — фиолетовый и так далее — см. [полную таблицу](#поддерживаемые-секции)), и названием секции на каждой карточке на выбранном языке вывода. Достаточно одного взгляда, чтобы понять, правильно ли песня разбилась на секции.

**Светлая и тёмная тема**, интерфейс на русском и украинском (о том, почему тут вообще нет сервера — см. [Приватность](#приватность)).

Сравнение регистронезависимое и работает как для названия, написанного просто на отдельной строке (`Припев`), так и в квадратных скобках (`[Припев]`).

## Опции форматирования

| Опция                  | Что делает                                                                                                                                                    |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Заглавные буквы**    | Делает заглавной первую букву каждого слайда (с учётом кириллицы — обрабатывает и `і`/`ї`/`є`/`ґ`).                                                           |
| **Удалить пунктуацию** | Убирает запятые, точки, кавычки, `«»`, многоточия и похожие знаки; строка, ставшая пустой после очистки, отбрасывается целиком, а не остаётся пустым слайдом. |
| **Удалить ссылки**     | Убирает из строки ссылки `http(s)://` и `www.`; такое же поведение с отбрасыванием пустой строки.                                                             |
| **Язык вывода**        | Выбирает язык названий секций (и `Title:`) — 🇬🇧 английский (по умолчанию), 🇷🇺 русский, 🇺🇦 украинский или 🇩🇪 немецкий. См. [Поддерживаемые секции](#поддерживаемые-секции). |

## Быстрый старт

Понадобится [Node.js](https://nodejs.org/) и npm.

```bash
git clone git@github.com:progeroffline/propresentor-formatter.git
cd propresentor-formatter
npm install
npm run dev
```

Откройте выведенный локальный адрес, вставьте текст песни и нажмите **Форматировать**.

## Добавление UI-компонентов

Проект использует shadcn/ui — новые компоненты добавляются через CLI, а не пишутся вручную, чтобы оставаться согласованными с настроенным стилем:

```bash
npx shadcn@latest add button
```

Импортируются как любой другой компонент:

```tsx
import { Button } from "@/components/ui/button"
```

## Приватность

Здесь нет бэкенда. Разбор текста, форматирование и отрисовка предпросмотра — всё происходит локально в вашем браузере: то, что вы вставляете, никуда не отправляется.
